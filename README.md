## What is IELTS Helper?

IELTS Helper is a Laravel a web application to connect English language learners with students who are seeking help from a deciated tutor. 


## Config and Setup

Once you have set up Composer and WSL and you have to install PHP and ensure that you are running Laravel 9 in order to run this project.
